# Ipon.ph
A local frontend project (HTML/JS/CSS) to mock and track personal Philippine bank savings.
Currently relies on `localStorage` for mocking sessions and balances, but structured to easily support a backend.

## Features
- Simulated user login and signup functionality
- Sleek interactive dashboard with modern UI/UX design
- Tracks mock balances for various Philippine banks (BDO, BPI, GCash, Maya, etc.)

## How to use
1. Open `login.html` in your browser.
2. Sign up to create a simulated account (saved in browser storage).
3. Test out adding dummy accounts and managing your balances on the dashboard.
